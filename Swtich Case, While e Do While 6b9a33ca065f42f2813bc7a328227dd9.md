# Swtich Case, While e Do While

**Usando switch case:**

```csharp
int valor = 1;
switch (valor)
{
	case 1: Console.WriteLine("1"); break;
	case 2: Console.WriteLine("2"); break;
	case 3: Console.WriteLine("3"); break;
	default: Console.WriteLine("4"); break; // Se n√£o for 1,2 ou 3
}
```
